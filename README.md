# ASLAlphabet image classification with PyTorch

The project contains jupyter notebook with code for training and evaluating CNN in PyTorch with purpose to correctly classify images of hand gestures to ASL (American Sign Language) alphabet.
The model's accuracy is more than 99%.

## Data Source
The dataset comes from Kaggle: https://www.kaggle.com/datasets/grassknoted/asl-alphabet/data?select=asl_alphabet_test

## Dependencies
- Opendatasets
- NumPy
- Pandas
- MatPlotLib
- PIL
- PyTorch
- Scikit-learn
